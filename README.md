# Hide Main Screen Buttons

You can also find this on the [Godot Asset Library](https://godotengine.org/asset-library/asset/4924).

Lets you hide the main screen buttons you do not want to see (2D, 3D, Script, Game, AssetLib) without disabling features.

You can control which buttons to hide in the Project Settings.

![](/assets/banner.png)

---

Godot allows you to [manage the editor features](https://docs.godotengine.org/en/stable/tutorials/editor/managing_editor_features.html).
You can use this to hide main screen buttons, but you will lose the associated features and the editor can then behave unexpectedly.

I want to hide some of these buttons without screwing up the editor's behaviour. So, I made this add-on.
